Composición Comida

La composición en fotografía son los elementos de una escena para hacerlos atractivos para el espectador. Estos elementos pueden representar el mensaje visual del fotógrafo de manera armoniosa, impactante y efectiva. en este caso se implemento para alimentos así llamando la atención del espectador a quererlos consumir o hacerlos bien desde casa, cuidando cada uno de los elementos en la composición
